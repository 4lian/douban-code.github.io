### CODE 是什么

CODE 是[豆瓣「Douban」](http://www.douban.com/)内部以工程师为主的代码管理和协作工具平台，功能类似 [GitHub](https://github.com/)，[Bitbucket](https://www.bitbucket.org/)，[GitLab](http://gitlab.org/)。

CODE 以 git 版本控制工具为基础，提供一个基于 Web 的仓库访问和协作的平台。
